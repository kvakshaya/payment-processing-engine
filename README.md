# Payment Processing Engine
A production-grade payment processing backend built with Java 17, Spring Boot, and PostgreSQL.

## Features
- REST APIs for payment initiation, status tracking, and reconciliation
- JWT-based authentication and role-based access control
- Idempotency key support to prevent duplicate transactions
- Kafka-based async event processing for payment notifications
- Reconciliation report generation with scheduled jobs

## Tech Stack
Java 17 · Spring Boot · Spring Security · Hibernate/JPA · PostgreSQL · Redis · Kafka · Docker · JWT
